# Preven
O Preven é um sistema de auxílio a prevenção de doenças ocupacionais em informática.
O Sistema procura mostrar que a solução de uma tarefa é mais saudável para o usuário se for divida em módulos e pausas.
O controle de tempo do uso do computador, por meio de pausas, estimula e orienta os usuários na execução de exercícios de revitalização.
Como funciona?
Este sistema monitora as atividades do usuário através da constante varredura da situação em que os periféricos de entrada, o teclado e mouse, se encontram.
A interrupção destes periféricos é feita após um determinado tempo (50 minutos), durante o qual estão sendo utilizados, 
permitindo que o repouso de 10 minutos seja realizado com uma frequência controlada. Durante o repouso são exibidas mensagens, exercícios de alongamento e relaxamento.
Após este estágio, as atividades são restauradas, podendo o usuário retornar ao trabalho.
Atividade - Atividade é o intervalo contínuo de 50 minutos para a realização de um trabalho ou parte dele.
Pausa - Pausa é o intervalo de 10 minutos entre os períodos de atividades no qual são exibidos exercícios de alongamentos prevenção às DOI.
Observações: O usuário não deve jamais ficar condicionado somente ao sistema, pois o seu corpo é que sabe exatamente a hora de parar.
Deve-se parar sempre que o sistema interromper as atividades, porém, se desejável, o repouso deve ser feito, mesmo que o sistema não o provoque automaticamente.
